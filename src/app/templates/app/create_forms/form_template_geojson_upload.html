Add new features

Instructions for file upload:

A *.geojson file containing multiple features can be uploaded. Any projection supported by GDAL can be used

Supported features: polygon, point, linestring.

Feature's attributes should contain following fields:

  • "name" - text value that will be assigned to a new feature (required) and
  • "type" - text value representing type of the created feature (required).
  • "sampled_feature" - name of the feature which is sampled be the created feature (optional).

    • "type" fields should contain one of the following: "sampling_feature:station", "sampling_feature:profile", "sampling_feature:area", "hydrography:stream", "hydrography:junction", "hydrography:lake", "hydrography:pond", "hydrography:reservior", "hydrography:watershed", "hydrography:water_divide", "hydrogeology:aquifer", "hydrogeology:aquitard", "hydrogeology:water_divide", "hydrogeology:layer", "hydrogeology:well", "other".

    "sampled_feature" fields should contain a name of the feature that is sampled by the new uploaded feature (e.g. name of a stream that is sampled by an uploaded station(s)). This field is not required and can remain empty.
